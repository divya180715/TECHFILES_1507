
 ATM Simulation Project
 Description

This project is a simple **ATM Simulation System** that mimics basic banking operations such as balance checking, cash withdrawal, deposit, and user authentication. It is designed to demonstrate fundamental programming concepts like control structures, functions, and user interaction.

The system provides a console-based interface where users can securely perform transactions similar to a real ATM.

---

 Features

* User authentication (PIN-based login)
* Balance inquiry
* Cash withdrawal
* Deposit functionality
* Transaction summary display
* Simple and user-friendly interface

 How to Run the Project

#### 🔹 Step 1: Clone the Repository

```bash

```

#### 🔹 Step 2: Compile the Code

```bash
g++ main.cpp -o atm
```

#### 🔹 Step 3: Run the Program

```bash
./atm
```

*(For Windows users)*

```bash

```
